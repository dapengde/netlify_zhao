How can we assess the relationship between the two? Correlation : The technique used to assess two variables that have a co-relationship with one another. That is, there is no meaningful way to decide which is the dependent and which is the independent variable. Positive correlation : High values on one are associated with high values on the other. Negative correlation : High values on one are associated with low values on the other. # Correlation Coefficient The parameter $\rho$ is the correlation coefficient which measures the strength of the linear relationship. The parameter $\rho$ varies between -1 and +1. If $\rho = âˆ’1$ there is a perfect negative correlation between the two variables, while $\rho = 1$ is a perfect positive correlation. If $\rho = 0$ the two variables are not linearly correlated. The sample correlation coefficient $r$ is: $$r = \frac{\frac{1}{N-1} \sum(x-\bar x)(y - \bar y)}{s_x s_y}$$ or: $$ r = \frac{\mathrm{cov}(x,y)}{s_x s_y}$$ $\mathrm{cov}(x,y)$ : covariance, which represents $x$ and $y$ varying together. !!! note: Exercise Calculate the correlation coefficient using the equations above. What does it mean? # Revisit the regression | Source | $df$ | $SS$ | $MS$ | $F$ | | ----------------------------------------- | -------------- | ------------------------------------------------------------ | --------------------------- | ----------------------- | | between the point and the mean | $df_\mathrm{total} = N - 1$ | $SS_\mathrm{total} = \sum(y -\bar y)^2$ | $MS_\mathrm{total} = \frac{SS_\mathrm{total}}{df_\mathrm{total}}$ | $F = \frac{MS_\mathrm{regression}}{MS_\mathrm{residual}}$ | | between the point and the regression line | $df_\mathrm{residual} = N - 2$ | $SS_\mathrm{residual} = \sum(y-y')^2$ | $MS_\mathrm{residual} = \frac {SS_\mathrm{residual}}{df_\mathrm{residual}}$ | | | between the regression line and the mean | $df_\mathrm{regression} = 1$ | $SS_\mathrm{regression} = \sum(y'-\bar y)^2$ | $MS_\mathrm{regression} = \frac{SS_\mathrm{regression}}{df_\mathrm{regression}}$ | | Alternatively, we compare $SS_\mathrm{regression}$ with $SS_\mathrm{total}$.
